Welcome to the **Department of Paranoia Login Portal**. This is a diabolical, zero-build React application designed to keep Muggles, Death Eaters, and casual fans entirely locked out of your system. 

It masquerades as a simple login page, but hides a punishing security protocol based on wildly fabricated, completely un-Googleable Harry Potter fan theories.

## ⚡ Features

* 🔒 **The First-Try Trap:** The system is fundamentally rigged. No matter what you answer on your very first attempt, the system will declare it incorrect. 
* ⏳ **Compounding Temporal Lockdown:** You get exactly 1 attempt. If you fail, the app locks down for 24 hours using local storage. Fail again tomorrow? It locks down for 48 hours. Then 72. 
* 🧠 **Un-Googleable Trivia:** The trivia questions are fresh, 100% fabricated, unhinged fan theories. Searching the internet will not save you.
* ⚛️ **Zero-Build React:** No Node.js, Webpack, or NPM required. The entire app runs out of a single `index.html` file using CDN-delivered React, Babel, and Tailwind CSS.

## 🛠️ How to Use & Deploy

Because this application uses in-browser Babel compilation, deployment is incredibly simple.

### Option 1: Run Locally
1. Download or copy the `index.html` file.
2. Double-click the file to open it in any modern web browser.
3. That's it. 

### Option 2: Host on GitHub Pages (Free)
1. Upload `index.html` to the root of a GitHub repository.
2. Go to your repository **Settings** > **Pages**.
3. Under **Build and deployment**, set the Source to **Deploy from a branch**.
4. Select the `main` (or `master`) branch and click **Save**.
5. Your paranoid portal will be live in 1-2 minutes at `https://[your-username].github.io/[repo-name]/`.
